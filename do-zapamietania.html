Skróty klawiszowe

By rozpocząć szkielet strony (pierwszy kod) w Visual Studio Code wciskamy „!” i „Tab”
Dzięki wciśnięciu klawiszy „Ctrl” i „S”, w programie Visual Studio Code zapisujemy zmiany na stronie klawiszami „Ctrl” i „F” włączamy wyszukiwanie danego elementu
„Ctrl” i „Z” cofamy ostatnie zmiany
„Ctrl” i „Y” przywracamy ostatnie zmiany
Klawisze „Ctrl” i „X” wycinamy element
„Ctrl” i „V” wklejamy elwment
„Alt” i „Z” zawijamy tekst w Visual Studio Code
Przycisk „Tab” to „ręczne” robienie wcięć lub, by ładnie sformatować wciskamy „Ctrl”, „Shift” i „P”. Potem w wyskakującą linijkę wpisać „format” i wybrać z opcji „format document”
„Ctrl” i „A” zaznaczamy cały tekst
„Ctrl” i „C” kopiujemy zaznaczony tekst
„Ctrl” i „R” lub „F5” odświeżamy stronę w przeglądarce
„Ctrl” i „Spacja” podpowiedź/ możliwe do użycia

Znaczniki

Jedne z części składowych kodu HTML. Ich budowa wygląda następująco:
(znacznik atrybut=”wartość”)treść(/znacznik) .
!DOCTYPE html - typ dokumentu
html lang="pl" - język strony tutaj polski
main - główna część dokumentu
section - sekcje dokumentu
article - samodzielne treści (artykuły)
header - nagłówek dokumentu/sekcji/artykułu
footer - stopka dokumentu/sekcji
Od (h1) do (h6) tekstowe nagłówki tytułowe- dokumentu/sekcji (tu chodzi o wielkość, ważność)
(p) paragraf/akapit
(link href=”nazwa.pliku” rel=”stylesheet” type=”text/css”) link do innego pliku
(br) nowa linia (jest pojedynczym znacznikiem)
(b) pogrubienie tekstu (efekt tylko wizualny)
(strong) pogrubienie tekstu (podkreśla istotność i powagę frazy)
(i) kursywa (pochylenie tekstu; efekt tylko wizualny)
(em) kursywa (podkreśla inne znaczenie np. ironię)
(a href=””target=””)opis łącza(/a) linki/łącza wewnątrz dokumentu
(element id=”mój identyfikator”) atrybut id (identyfikuje element; nawigacja w obrębie dokumentu)
np. (a href=”#notatki”)Przejdź do notatek(/a) lub (a href”#top”)Wróć na górę strony(/a)
(nav) nawigacja
(img src=”” alt=””) img- obrazek o podanym adresie
(znacznik style=”color:yellow;”)treść(/znacznik) styl przy znaczniku z wykorzystaniem atrybutu style nazywamy stylem inline
(znacznik class=”name”)treść(/znacznik) klasa; jak tworzymy nazwę (name) wykorzystujemy kebab-case; tutaj używamy jeszcze .name{
właściwość: wartość;
}
(znacznik id=”name”)treść(/znacznik) tutaj używamy #name{
właściwość: wartość;
}
!important „zabija” wszystkie inne reguły; dopisuje się go pomiędzy „wartość”, a „ ; ” gdy piszemy style klasy. Zazwyczaj się tego nie używa, bo mąci w kodzie.
Gdy chcemy stworzyć skrypt na stronie (np. przywitanie), za pomocą JavaScript używamy: (script),
alert('tekst');
(/script)
a potem możemy to przerzucić do pliku z JavaScript’em za pomocą: (script src="main.js")(/script)
console.log(‘tekst’); tworzenie skryptu (np. przywitania) w bardziej sekretny sposób, otóż w konsoli
const name = 'Jola'; zmienne skryptu. Tak podajemy imię, a tak wiek const age = 21; zamiast name można wpisać np. firstName. Trzeba pamiętać, że do tego używamy camelCase’a
console.log(name); by wrzucic (w tym wypadku) name do konsoli na stronie.
console.log(`Hejka nazywam się ${name} i mam ${age} lat.`); w ten sposób też można się przedstawić w konsoli

Polecenia w terminalu:

dir – wypisanie zawartości katalogu
cd nazwa-katalogu – przejście do katalogu o nazwie „nazwa-katalogu”. gdy chcemy przejść do (np.) głównego katalogu dysku C piszemy cd C:\. Klawiszem „Tab” wybieramy pomiędzy katalogami zawartymi w miejscu, w którym jesteśmy.
cd – wyświetlanie aktualnej ścieżki
code . – tak z poziomu terminala odpalamy Visual Studio Code (pamiętamy o spacji pomiędzy code oraz . ) w aktualnym katalogu
cd .. – przenosi do katalogu wyżej
mkdir nazwa-katalogu – stworzenie katalogu o nazwie „nazwa-katalogu”
